# AI Companion Chatbot

An AI-powered conversational companion chatbot built with FastAPI and OpenRouter.

## Features
- Personality-based conversations
- Context memory
- FastAPI backend
- OpenRouter LLM integration

## Tech Stack
- Python
- FastAPI
- OpenRouter API
- Git/GitHub

## Run Locally

```bash
pip install -r requirements.txt
uvicorn main:app --reload
